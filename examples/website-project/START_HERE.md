# Start Here

This is the operating workspace for the Acme website rebuild.

## What this project is

Acme is rebuilding its public website to match its new brand positioning. The project involves landing page design, component reuse, copy and positioning work, and multi-locale deployment.

## Mandatory reading

1. `START_HERE.md` (this file)
2. `AGENTS.md`
3. `STATUS.md`

## Additional context

- `DOMAIN.md` - brand rules, design system, repo structure
- `BENCHMARKS.md` - what "good" means for each page
- `DECISIONS_LOG.md` - key design and implementation decisions
- `templates/LANDING_PAGE_BUILDER.md` - prompt for agents building new landing pages
- `templates/BRAND_CONSISTENCY_AUDITOR.md` - prompt for brand consistency reviews
- `reference/` - brand documents, competitor screenshots, design specs

## File stability

Stable core: `START_HERE.md`, `AGENTS.md`, `DOMAIN.md`, `BENCHMARKS.md`
Evolving edge: `STATUS.md`, `DECISIONS_LOG.md`, `templates/`, `reference/`
